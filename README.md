# 简化版无人机信息管理系统

这是重新生成的简化版项目，保留最核心功能：

- 登录
- 查询无人机列表
- 新增无人机
- 修改无人机
- 删除无人机
- 新增时自动生成无人机属性
- 默认使用 SQLite，可切换 MySQL

## 启动方式

1. 双击 `start-backend.bat` 启动后端。
2. 双击 `start-frontend.bat` 启动前端。
3. 浏览器打开：

```text
http://localhost:5174
```

默认账号：

```text
admin / admin123
```

后端接口地址：

```text
http://localhost:8092/api
```
